    smcx.types

    smcx.types

    Shared aliases and callback protocols for smcx.

    Core array and key aliases follow the conventions used by Dynamax (dynamax.types); callback protocols describe smcx's public boundaries.

    FilterCarry = PyTree[Shaped[Array, '...']] module-attribute

    Caller-owned JAX PyTree carried by a particle-filter kernel.

    PRNGKeyT = PRNGKeyArray module-attribute

    JAX PRNG key (handles both old and new JAX key formats).

    ParticleCloud = PyTree[Shaped[Array, 'num_particles ...']] module-attribute

    Latent-state PyTree with a leading particle axis on every leaf.

    ParticleHistory = PyTree[Shaped[Array, 'ntime num_particles ...']] module-attribute

    Latent-state PyTree with leading time and particle axes.

    Scalar = float | Float[Array, ''] module-attribute

    Python float or scalar JAX array with float dtype.

    StateHistory = PyTree[Shaped[Array, 'ntime ...']] module-attribute

    Single-trajectory state PyTree with a leading time axis.

    StateTree = PyTree[Shaped[Array, '...']] module-attribute

    One latent state represented by a nonempty JAX PyTree of arrays.
